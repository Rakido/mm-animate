# Transformation

import NxTextAnimation from '../../components/NxTextAnimation'

## Rotation

L'attribut `data-rotate` donne une rotation de départ aux éléments :

```html
<div 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-stagger="0.1"
  data-axis="-y"
  data-rotate="40"
>
  Les caractères s'animent avec 40° de rotation au départ
</div>
```

<NxTextAnimation 
  data-scroll-text-reveal

  data-animate="fade-in"
  data-stagger="0.1"
  data-rotate="40"
>
  Les caractères s'animent avec 40° de rotation au départ
</NxTextAnimation>

## Scale

L'attribut `data-animate="scale"` permet d'animer les caractères, mots ou lignes depuis un scale de 0 à 1 :

```html
<div 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="scale"
  data-stagger="0.015"
  data-easing="elastic.out"
>
  Les caractères s'animent avec un scale de 0 à 1
</div>
```

## Skew

L'attribut `data-skew` donne une inclinaison de départ aux éléments :

```html
<div 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-stagger="0.1"
  data-skew="40"
>
  Les caractères s'animent avec 40° d'inclinaison au départ
</div>
```

<NxTextAnimation 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="fade-in"
  data-stagger="0.1"
  data-skew="40"
>
  Les caractères s'animent avec 40° d'inclinaison au départ
</NxTextAnimation>

---

## Combinaison

Les deux attributs `data-rotate` et `data-skew` peuvent être utilisés ensemble :

```html
<div 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-stagger="0.1"
  data-skew="90"
  data-rotate="20"
>
  Les caractères s'animent avec 40° d'inclinaison et 20° de rotation
</div>
```
