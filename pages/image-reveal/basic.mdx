# Les bases de l'Image Reveal

import NxImageReveal from '../../components/NxImageReveal'

## Import

La librairie est disponible ici : [MM-ImageReveal.js](https://raw.githubusercontent.com/Rakido/mm-animation-library/refs/heads/main/js/mm-image-reveal.js)

Pour utiliser l'animation de révélation d'image, vous aurez besoin de trois dépendances :

- [GSAP](https://www.jsdelivr.com/package/npm/gsap)
- [GSAP Custom Easing](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/CustomEase.min.js)
- [GSAP ScrollTrigger](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js)

## Utilisation Basique

Pour créer une animation de révélation d'image, vous aurez besoin de deux attributs principaux :
- `data-scroll-image-reveal` sur le conteneur

## Configuration de base

```html
<div data-scroll-image-reveal class="reveal">
  <img src="/path/to/image.jpg" />
</div>
```

### Nécessite du CSS
```css
.reveal {
  position: relative;
  overflow: hidden;
}

.reveal img {
  display: block;
  width: 100%;
  height: auto;
}
```
