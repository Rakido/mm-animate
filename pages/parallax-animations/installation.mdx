# Installation

import NxParallaxAnimation from '../../components/NxParallaxAnimation'

## Import

La librairie est disponible ici : [MM-ParallaxAnimation.js](https://raw.githubusercontent.com/Rakido/mm-animation-library/refs/heads/main/js/mm-parallax-animation.js)

## Dépendances
Pour utiliser l'animation de parallax d'image, vous aurez besoin de trois dépendances et un fichier CSS:

- [MM-ParallaxAnimation.css](https://github.com/Rakido/mm-animation-library/blob/main/css/data-parallax.css)
- [GSAP](https://www.jsdelivr.com/package/npm/gsap)
- [GSAP Custom Easing](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/CustomEase.min.js)
- [GSAP ScrollTrigger](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js)


## Animation de base

Le parallax permet d'animer des images avec un effet de profondeur au scroll.

Pour créer une animation parallax, vous aurez besoin de :
- `data-parallax` sur le conteneur
- `data-image-parallax` sur le conteneur

```html
<div class="parallax-container"
     data-parallax 
     data-image-parallax >
  <img 
    src="image.jpg" 
  />
</div>
```

<NxParallaxAnimation 
  src="https://picsum.photos/1200/800"
  imageFull
  data-parallax
  data-image-parallax
/>
