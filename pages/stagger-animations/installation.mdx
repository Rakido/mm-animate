# Installation

import NxStaggerAnimation from '../../components/NxStaggerAnimation'

Le stagger permet d'animer des éléments en séquence avec un délai entre chaque élément.

## Import

La librairie est disponible ici : [MM-StaggerAnimation.js](https://raw.githubusercontent.com/Rakido/mm-animation-library/refs/heads/main/js/mm-stagger-animation.js)

Pas de CSS associé pour ce bloc.

## Dépendances

Pour utiliser les animations stagger, vous aurez besoin de deux dépendances :

- [GSAP](https://www.jsdelivr.com/package/npm/gsap)
- [GSAP Custom Easing](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/CustomEase.min.js)
- [GSAP ScrollTrigger](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js)

## Configuration de base

Pour créer une animation stagger, vous aurez besoin de :
- `data-stagger-reveal` sur le conteneur parent


```html
  <ul data-stagger-reveal>
    <li></li>
    <li></li>
    <li></li>
    ...
  </ul>
```

Cela animera chaque enfant à l'intérieur.

<NxStaggerAnimation data-stagger-reveal>
  {[1, 2, 3, 4].map((item) => (
    <div 
      key={item}
      data-stagger-item
      className="nx-bg-gray-700 nx-p-4 nx-rounded-lg nx-mb-4"
    >
      Item {item}
    </div>
  ))}
</NxStaggerAnimation>
